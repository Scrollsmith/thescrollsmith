# THE SCROLLSMITH Website

Custom static site (no WordPress) built for a mythic, digital, ascendant brand style.

## Pages

- `index.html` - Home
- `music.html` - Music and audio players
- `kyros.html` - Worldbuilding page
- `codex.html` - Lore structure
- `projects.html` - Extra works
- `contact.html` - Contact and links

## Asset Placement

Put your files in:

- `assets/img/cover-main.jpg` (home featured cover)
- `assets/img/kyros-visual.jpg` (Kyros page visual)
- `assets/audio/ascendant-gate.mp3`
- `assets/audio/iron-prophecy.mp3`
- `assets/audio/glyph-engine.mp3`

You can rename files if you also update the matching paths in HTML.

## Local Preview

Open `index.html` directly, or run a local server:

```powershell
python -m http.server 8080
```

Then visit: `http://localhost:8080`

## Deploy

Upload all files/folders to your hosting root (often `public_html` or `www`).
